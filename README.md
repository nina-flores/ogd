# ogd

# Residential proximity to oil and gas development and socio-economic conditions: A comparative analysis of 33 countries
Authors: Lara Schwarz, Nina Flores

This repository contains scripts used to study residential proximity to oil and gas development and socio-economic conditions across 33 countries. 

# Data Sources
Data from this analysis comes from multiple sources:
* Oil and Gas Infrastructure Mapping (OGIM) Database v2.5.1
* Global Human Settlement Layer, a 5-year estimate of the residential population
* Institute for Health Metrics and Evaluation Social data
* Global Relative Deprivation Index (GRDI)


**Environmental variables**
-  oil and gas wells

**Social variables**
- lag-distributed income (LDI)
- Global Relative Deprivation Index (GRDI)
